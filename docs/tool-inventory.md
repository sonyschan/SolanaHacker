# Agent Tools 完整清單

> SolanaHacker Agent 可用工具的完整盤點

---

## 📋 完整工具清單 (按字母順序)

### A-C
1. **`add_bug_solution`** - 新增錯誤解決方案到記憶庫
2. **`browse_url`** - 瀏覽外部 URL 並進行視覺分析
3. **`check_console_errors`** - 檢查瀏覽器控制台錯誤 (節省 token)
4. **`colosseum_project`** - 管理 Colosseum 競賽專案提交
5. **`complete_task`** - 標記當前任務為完成狀態

### D-G  
6. **`dev_server`** - 控制 Vite 開發伺服器
7. **`git_commit`** - 本地 Git 提交 (不推送)
8. **`git_commit_push`** - Git 提交並推送到遠端 (已棄用)
9. **`git_release`** - 推送並建立版本標籤

### L-R
10. **`list_files`** - 列出目錄中的檔案和資料夾
11. **`load_skill`** - 載入專業技能模組 (4個可選)
12. **`log_attempt`** - 記錄工作嘗試到日誌
13. **`lookup_bug`** - 在記憶庫中查找錯誤解決方案
14. **`read_file`** - 讀取檔案內容
15. **`read_knowledge`** - 讀取參考知識庫
16. **`remember`** - 新增項目到長期記憶
17. **`request_approval`** - 請求人工審核批准
18. **`review_ux`** - 執行完整的 UX 評估
19. **`run_command`** - 執行 shell 命令

### S-W
20. **`search_memory`** - 搜索長期記憶庫
21. **`send_telegram`** - 發送 Telegram 通知
22. **`take_screenshot`** - 擷取應用程式截圖
23. **`update_current_task`** - 更新當前任務狀態 (限制使用)
24. **`write_file`** - 寫入檔案內容
25. **`write_journal`** - 寫入每日工作日誌

---

## 📊 統計摘要

- **核心檔案操作**: 5 個
- **開發環境管理**: 4 個  
- **記憶和學習**: 6 個
- **專案管理**: 5 個
