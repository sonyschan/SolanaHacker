## 🎯 Product Overview

**MemeForge** 是一個 Web3 平台，結合 AI 生成的梗圖、社群投票和經濟激勵，創建一個民主化的 NFT 市場，讓用戶通過參與決定價值。

### Core Value Proposition
- **免費參與**: 投票不需任何費用
- **真實獎勵**: 贏得實際的 SOL 獎金
- **民主定價**: 社群投票決定 NFT 稀有度和價值  
- **持續參與**: 週期性獎勵系統保持用戶活躍
- **稀缺性驅動**: 每日僅鑄造 1 個 NFT，創造真正稀缺性
- **公平機制**: Random Ticket Rewards 防止投票操控
- **100% 社群決定**: 完全由社群決定稀有度，無平台干預

## 🔄 完整價值循環 (6-Step Business Model)

```
1. AI 生成 Meme
    ↓ (自動化內容生產)
2. 用戶投票 → 獲得彩票 (8-15張/次)
    ↓ (免費參與，獲得獎勵機會)
3. 投票結果決定 Meme 稀有度 (Common → Legendary)
    ↓ (民主化價值發現)
4. 高稀有度 Meme → 鑄造為 NFT → 競標拍賣
    ↓ (稀有內容變現)
5. 競標收益 → 進入獎池 (80% 分配給用戶，20% 用於營運)
    ↓ (創造實際價值)
6. 週日 8PM UTC 開獎 → 分配 SOL 給中獎者
    ↓ (價值回饋參與者，激勵下週參與)
```

### 詳細步驟說明

#### Step 1: AI 生成 Meme
- **流程**: AI 自動生成獨特的梗圖內容
- **頻率**: 持續生成，每日 3 個新梗圖
- **品質**: 經過幽默度和病毒傳播潛力篩選
- **內容來源**: Twitter 趋势、CoinDesk 新聞、Reddit r/CryptoCurrency、政治事件、技術新聞
- **技術**: Gemini API (聊天使用 Grok，圖像生成使用 Gemini)
- **成本**: 單次生成約 $0.1-0.3

#### Step 2: 用戶投票 (核心互動) - 兩階段投票系統
- **第一階段**: 用戶從 3 個梗圖中選擇最喜歡的 (勝者投票)
- **第二階段**: 對勝出梗圖投票決定稀有度 ("讚" 或 "踩")
- **費用**: 完全免費，無 Gas Fee
- **獎勵**: 每次完整投票獲得 8-15 張彩票
- **連勝獎勵**: 連續每日投票的額外獎勵機制
- **時間**: 每個投票期 24-48 小時
- **要求**: 僅需連接錢包，無需代幣

#### Step 3: 決定稀有度 (價值發現)
- **輸入**: 投票期結束後的投票匯總
- **輸出**: 稀有度分類 (Common → Legendary)
- **演算法**: 讚踩比例 + 參與度指標
- **透明度**: 結果公開可查
- **客觀性**: 自動化計算，無人為干預

#### Step 4: 鑄造 NFT (價值實現) - 稀缺性驅動
- **每日限制**: 僅鑄造 1 個 NFT (最高稀有度獲勝者) - 創造真正稀缺性
- **流程**: 自動鑄造到 Solana 區塊鏈
- **拍賣**: 3 天拍賣期，最高價得標
- **拍賣規則**: 起標價 0.01 SOL，最小加價 5%，自動退款機制
- **平台**: 內建市場，無需外部平台
- **安全**: Multi-sig wallet 管理，Smart contract audits

#### Step 5: 獎池累積 (價值聚合)
- **來源**: NFT 拍賣收益 (80% 進入獎池，20% 用於營運)
- **累積**: 每週獎池，透明追蹤
- **安全**: Multi-sig wallet 管理，資金安全保障
- **增長**: 隨 NFT 銷售額增長
- **公平遊戲**: Fair play 機制確保公正分配

#### Step 6: 分配獎勵 (價值回饋)
- **時程**: 每週日晚上 8 點 UTC
- **方式**: Random Ticket Rewards - 基於持有彩票數量的公平抽獎
- **貨幣**: SOL 獎勵直接發放到錢包
- **驗證**: 鏈上分配證明，完全透明
- **防操控**: 隨機抽籤機制防止投票系統被操控

## 🏆 稀有度系統 & 經濟模型

### 稀有度級別

| 稀有度 | 出現機率 | 判定標準 | 投票者彩票獎勵 | NFT 鑄造 | 拍賣起標價 |
|--------|----------|----------|----------------|----------|------------|
| **Common** | ~45% | 讚踩比例 < 60% | 8-15 張彩票 | ❌ 不鑄造 | - |
| **Uncommon** | ~30% | 讚踩比例 60-70% | 8-15 張彩票 | ❌ 不鑄造 | - |
| **Rare** | ~15% | 讚踩比例 71-80% | 8-15 張彩票 | ⭐ 競爭鑄造* | 0.01 SOL |
| **Epic** | ~8% | 讚踩比例 81-90% | 8-15 張彩票 | ⭐ 競爭鑄造* | 0.01 SOL |
| **Legendary** | ~2% | 讚踩比例 > 90% | 8-15 張彩票 | ⭐ 競爭鑄造* | 0.01 SOL |

### 重要機制說明：
- **每日限制**: 僅鑄造 1 個 NFT (當日最高稀有度獲勝者)
- **同級競爭**: 相同稀有度時，以投票參與數量決定
- **彩票機制**: 所有投票者獲得相同數量彩票 (8-15張)，稀有度不影響彩票數量
- **獎勵來源**: 週日開獎的 SOL 獎勵來自 NFT 拍賣收益的 80%
- **判定透明**: 讚踩比例 + 參與度指標，自動化計算，結果公開可查

### 經濟機制詳解

#### 收入流
1. **NFT 拍賣手續費**: 所有銷售額的 20% 平台費
2. **優質功能費**: 可選的外觀/徽章 (未來功能)
3. **合作夥伴收入**: 品牌合作 (未來功能)

#### 成本結構
- **AI 生成成本**: 梗圖創作的 API 費用
- **Solana 交易費**: NFT 鑄造的最小費用
- **基礎設施**: 主機、CDN、資料庫
- **獎池分配**: NFT 拍賣收益的 80% 分配給用戶，20% 用於營運

#### 用戶經濟學
- **投票成本**: 免費
- **中獎機率**: 基於持有彩票數 ÷ 總彩票池
- **平均彩票價值**: 獎池總額 ÷ 總彩票數
- **期望報酬**: 對活躍參與者為正值

## 🛠️ 技術架構

- **框架**: React 18 + Vite (快速開發環境)

- **錢包整合**: Solana Wallet Adapter
- **狀態管理**: React Context + Zustand
- **UI 元件**: 自定義響應式設計
- **部署**: Vite build + 自訂服務器

### 後端服務
- **區塊鏈**: Solana mainnet/devnet
- **錢包連接**: Phantom, Solflare, Torus, MathWallet 支援
- **NFT 標準**: Metaplex Token Metadata Program

- **梗圖生成**: Gemini API (gemini-2.5-flash-image 用於 UX 資產，gemini-3-pro-image-preview 用於 NFT 藝術)
- **聊天/新聞**: Grok API 用於趨勢分析和新聞摘要
- **內容來源**: Twitter 趨勢、CoinDesk、Reddit r/CryptoCurrency 等
- **內容審核**: 自動化過濾
- **品質控制**: 參與度預測模型

## ⚠️ 風險分析 & 緩解措施

### 技術風險
- **風險**: Solana 網路壅塞影響用戶體驗
- **緩解**: 實施重試機制，用戶溝通
- **風險**: 智能合約漏洞
- **緩解**: 主網前專業審計，漸進式推出

### 經濟風險
- **風險**: NFT 銷售不足以維持獎池
- **緩解**: 動態獎勵調整，替代收入流
- **風險**: 投票系統的遊戲化/操控
- **緩解**: 錢包驗證、反機器人措施、速率限制

### 市場風險
- **風險**: NFT 市場衰退影響需求
- **緩解**: 專注實用性而非投機、社群建設
- **風險**: 既有平台的競爭
- **緩解**: 獨特價值主張、優質用戶體驗

### 監管風險
- **風險**: 潛在的賭博法律含義
- **緩解**: 法律審查、基於技能的元素、合規監控
- **風險**: 代幣獎勵的證券法規
- **緩解**: 法律結構審查、專注實用性而非投資

```
  🏆 Rarity determined by community votes
           ↓
 💎 Highest rarity → Daily 1 NFT minting
           ↓
  🔨 3-day auction (0.01 SOL start, 5% min bid)
           ↓
 💰 80% revenue → Prize Pool, 20% → Operations
           ↓
🎰 Sunday 8PM UTC → SOL distribution
           ↓
🔄 Winners reinvest, cycle repeats
```

### 循環強化機制
- **正向反饋**: 更多參與 → 更大獎池 → 更多新用戶
- **品質提升**: 更多投票 → 更準確稀有度 → 更高 NFT 價值
- **社群效應**: 中獎分享 → 口碑傳播 → 用戶增長
- **內容豐富**: 更多用戶 → 更多投票數據 → 更好的 AI 調優
