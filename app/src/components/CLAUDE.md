## Cross-Tab Navigation

- Tab switching is controlled by `setActiveTab` in `Dashboard.jsx`
- Child components that need cross-tab CTAs receive `setActiveTab` as a prop
- Current components with this prop: WorkshopTab, ForgeTab, LotteryTab

## CTA Banner Pattern

Standard CTA banners between sections use this structure:
- Full-width button with `bg-gradient-to-r from-{color}/20 to-{color}/20`
- Border: `border border-{color}/30 hover:border-{color}/50`
- Layout: flex row with icon (text-2xl), title (font-bold) + description (text-xs text-gray-400), and right chevron arrow
- Group hover: title color shift + chevron `translate-x-1`
