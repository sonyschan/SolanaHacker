# v0 UI Requirements: MemeForge Dashboard Redesign

## Purpose
Modern dashboard-centric redesign of MemeForge to achieve 90%+ UX score. Transform confusing crypto meme voting platform into clear, engaging, trustworthy dashboard application.

## Target Users
- Crypto-curious individuals interested in earning SOL
- Meme enthusiasts wanting community content curation
- Web3 natives looking for earning opportunities

## Key Features to Showcase
1. **Daily AI Meme Generation** - Fresh content every 24 hours
2. **Community Voting System** - Democratic winner selection  
3. **SOL Cryptocurrency Rewards** - Real earnings via lottery
4. **Transparent Lottery System** - Clear fair mechanics
5. **Streak Rewards** - Bonus tickets for consistency

## Design Requirements

### Framework & Styling
- **Framework**: React + Tailwind CSS
- **Style**: Modern Dashboard-Centric Design
- **Theme**: Dark mode with purple/blue gradient accents
- **Layout**: Dashboard grid with cards and panels

### Visual Hierarchy (Priority Order)
1. **Clear Value Proposition** (hero section)
2. **How It Works** (simple 3-step process)
3. **Daily Meme Voting Interface** (main content)
4. **Stats Dashboard** (user/community metrics)
5. **Wallet Connection** (lower friction)

### Dashboard Components Needed
- **Header**: Navigation with logo, stats, wallet button
- **Hero Section**: Clear value prop + "Start Voting" CTA
- **Stats Dashboard**: 4-6 key metrics in cards
- **Voting Interface**: Today's memes with voting buttons
- **Activity Feed**: Live community activity
- **How It Works**: 3-step process explanation

### Color Palette
- **Background**: Dark grays (#1a1a1a, #2d2d2d)
- **Accent**: Purple to blue gradients (#8b5cf6 to #3b82f6)
- **Success**: Green for earnings (#10b981)
- **Cards**: Dark with subtle borders

### Key Interactions
- Large, prominent "VOTE NOW" buttons
- Smooth hover animations on cards
- Clear loading states
- Mobile-responsive touch targets
- Progressive disclosure of complex concepts

### Mobile Optimization
- Touch-friendly buttons (44px+ minimum)
- Readable typography (16px+ on mobile)
- Simplified navigation for small screens
- Fast loading optimized layouts
