## Journal Format

All journal entries written to `memory/journal/memeya/{date}.md` must use the standard format:
```
## HH:MM:SS
- Topic: {topic_name}
- Posted: {text}
```
- Time must be GMT+8: `new Date(Date.now() + 8 * 3600_000).toISOString().slice(11, 19)`
- NEVER prefix the `##` header with words (e.g. `## Moltbook 17:52:26`) — the Workshop feed parser expects the timestamp first
- Always call `syncToBackend(baseDir)` after writing to the journal so entries appear in the Workshop feed immediately

## Workshop Feed Guidelines

- Use **template-based personality text** (not Grok/LLM-generated) for Moltbook log entries
- Grok-generated ambient logs are only for entries with no structured data to display
- Each Moltbook activity type has its own topic and template pool:
  - `moltbook_post` — posting memes, cross-posts, ecosystem posts, introductions
  - `moltbook_engage` — upvotes, comments, replies, DMs
- Templates should incorporate real data (title, submolt, engagement counts) in Memeya's voice
