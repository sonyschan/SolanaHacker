# Skill: grok_research

Web search, research documentation, and devlog writing with Grok AI.

## Trigger Hints

- search web
- research topic
- find information
- look up
- trending
- write research

## Tools

- `web_search`: Search the web for latest information
- `write_research`: Document research findings to docs/research_summary.md
- `write_devlog`: Generate casual devlog message in Chinese
