# Skill: xai_analysis

Analyze X/Twitter accounts and tokens with X.AI (Grok). Evaluate credibility, social presence, and trading potential.

## Trigger Hints

- analyze X account
- twitter account
- token analysis
- evaluate token
- social presence
- credibility check

## Tools

- `analyze_x_account`: Analyze X/Twitter account for credibility and influence
- `analyze_token`: Deep analysis of a Solana token
- `evaluate_social_presence`: Evaluate social media presence across platforms
