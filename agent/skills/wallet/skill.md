# Skill: wallet

Check Memeya's USDC reward wallet balance, view distribution history, and manually trigger reward payouts.

## Trigger Hints
- When checking wallet balance or USDC funds
- When reviewing reward distribution history
- When manually triggering reward payouts
- wallet, balance, rewards, USDC, payout

## Tools
- `wallet_balance` — Check Memeya's wallet USDC/SOL balance and thresholds
- `wallet_rewards` — View recent reward distribution history
- `wallet_distribute` — Manually trigger reward distribution for today's draw
